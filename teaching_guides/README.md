# Gator AI Camp 2026 - Teaching Guides

This folder contains step-by-step teaching guides that walk through all learning outcomes outlined in the main README, complete with references to the lines of code that need to be examined, modified, or added.

## Overview

The teaching guides are organized by week and day, matching the camp schedule:

### Week 1: Introduction to Python Programming
- **Day 1**: Python Setup and Game Introduction
- **Day 2**: Sprites and Game Logic
- **Day 3**: Dialogue Trees and Interactions
- **Day 4**: Advanced Dialogue and Debugging
- **Day 5**: Project Finalization and Presentations

### Week 2: Introduction to Artificial Intelligence
- **Day 1**: AI Concepts and Environment Setup
- **Day 2**: Facial Recognition Model Training
- **Day 3**: Large Language Models and APIs
- **Day 4**: Game Integration - Facial Recognition + LLM
- **Day 5**: Final Demonstrations and Reflection

## How to Use These Guides

Each day's guide includes:
1. **Learning Objectives**: What students should accomplish
2. **Core Concepts**: Key programming and AI concepts covered
3. **Hands-On Exercise**: Practical activity to reinforce learning
4. **Code References**: Specific files and line numbers to examine/modify
5. **Key Learning Points**: Summary of what students should understand
6. **Extension Activities**: Optional challenges for advanced students
7. **Troubleshooting Tips**: Common issues and solutions

## Navigation

- [Week 1 Guides](./Week1/)
- [Week 2 Guides](./Week2/)

Each guide is designed to be self-contained and can be followed independently, though they build upon each other throughout the two-week camp.

## Connection to Main Project

All code references in these guides point to files in the main GatorAI_Camp_2026 repository. Students should make modifications to their local copies of these files as they work through the exercises.

## Prerequisites

Before starting the camp, ensure you have:
1. Python 3.8+ installed
2. Git installed (for version control)
3. A code editor (VS Code recommended)
4. Internet access for downloading libraries and resources
5. A webcam (for AI exercises in Week 2)

## Getting Started

1. Clone the main repository: `git clone https://github.com/UFResearchComputing/GatorAI_Camp_2026`
2. Navigate to the project directory
3. Follow the Day 1 guide to set up your environment
4. Progress through each day's guide in order

Happy coding!
