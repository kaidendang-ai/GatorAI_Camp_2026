# Week 2, Day 3 & 4: LLM Integration

Time to connect to the Large Language Model!

## Step 1: Add Your API Key safely
Open `ai_dialogue_manager.py` and find `@STUDENT-EDIT-Week2_Day3-1`.
Add your API key safely in the `ai_materials/navigator_api_key.json` file. Remember, don't commit it to GitHub!

## Step 2: Modify the System Prompt
Find `@STUDENT-EDIT-Week2_Day4-1` in `ai_dialogue_manager.py`.
Modify the system prompt to give the AI a unique personality. Tell it to act like a pirate, a robot, or a grumpy farmer!

## Step 3: Custom Fallback Responses
Find `@STUDENT-EDIT-Week2_Day4-2`.
Create custom fallback responses for when the API is down or you lose internet connection.
