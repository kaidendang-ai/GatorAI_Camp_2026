# Day 1: Setup & Introduction

Welcome to PyDew Valley! Today we'll learn about Python fundamentals and how our game is set up.

## Step 1: Examine Datatypes and Add Comments
Open `settings.py` and look around line 13.
Look for `@STUDENT-EDIT-Day1-1`.
Identify the datatypes (strings, integers, lists) being used.
Write a comment using `#` to describe what a variable does.
Example:
```python
# The width of the screen is 1280 pixels
SCREEN_WIDTH = 1280
```

## Step 2: Customize the Game Window Title
Open `settings.py` and find `@STUDENT-EDIT-Day1-2`.
Change the `TITLE` variable to a string of your choice. Make it your own game!
```python
TITLE = "My Awesome Farm Game"
```

## Step 3: Change the Window Size
Open `settings.py` and find `@STUDENT-EDIT-Day1-3`.
Try changing `SCREEN_WIDTH` or `SCREEN_HEIGHT`. Notice how the window changes when you run the game.

## Step 4: Experiment with Background Colors
Open `settings.py` and find `@STUDENT-EDIT-Day1-4`.
Change `WATER_COLOR` to a different hex code like `"#FF0000"` for red or `"#00FF00"` for green.

## Step 5: Insert a Print Statement
Open `main.py` and find `@STUDENT-EDIT-Day1-5` inside the `__init__` method.
Insert a print statement to see when the game starts.
```python
print("Game starting!")
```
Check your console output when you run the game!
